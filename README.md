# Recipe Ecommerce Microservices

Spring Boot microservices architecture for a recipe-based ecommerce platform.

## Tech Stack
Spring Boot  
Spring Cloud Eureka  
Spring Cloud Gateway  
REST APIs  
Java  
Maven  

## Microservices
- Service Registry (Eureka)
- API Gateway
- User Service
- Recipe Service
- Order Service
- Payment Service
- Inventory Service
- Notification Service
- Admin Service

## Architecture

Client → API Gateway → Microservices  
                ↓  
          Service Registry (Eureka)
